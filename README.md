# UI Foundry Toast Studio

Standalone, self-contained Toast component generator built for marketplace sale and future UI Foundry SaaS merge-readiness.

## Features

- Status model: severity, title, body, icon, actions, persistence and dismissal
- Overlay behavior where native: modal/non-modal, focus return, escape/outside dismissal
- Placement: side, alignment, offset, collision handling, viewport padding
- Accessibility: live region, role, aria-modal, labels, reduced motion
- React-only export: preview, visible code, copy, and download are sourced from one payload.
- Local shared controls: no runtime imports from other component projects.

## Commands

```bash
npm install
npm run typecheck
npm run lint -- --quiet
npm run build
npm run dev
```
